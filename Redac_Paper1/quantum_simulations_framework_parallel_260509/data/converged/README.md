# data/converged/
> Last updated: 2026-05-11

Valid L=10, K=10 simulation outputs go here as HDF5 (.h5) files.

**Do not commit .h5 files without Git LFS configured.**

Files are generated by running:
```bash
mamba run -n MesoHOP-sim python reproducibility/main.py
```

CSV convergence audit results go to `reproducibility/results/`.
